Exercice 0 : Mon compte-rendu est trop stylé !

  1. J'ai ajouté à la balise <h2> un attribut style qui permet de mettre le titre de l'exercice en orange.
  2. Mon document utilise la police sans-serif.
  3. J'ai changé la taille des balises <h1>, <h2> et <h3>.

    h1 {
     font-size: 24pt;
    }
    h2 {
     font-size: 18pt;
    }
    h3 {

     font-size: 18pt;

    .centered {
     text-align: center;
    }
    ...

  6. J'ai aligné mon nom à droite avec la classe right.

    div.right {
     text-align: right;
    }
    ...
    <div class="right">Sarah OUCHEMOUKH</div>

  7. J'ai utilisé les pseudo-classes des liens pour changer la couleur du lien en fonction de son état.

    a:link {

Exercice 1 : Feuille de style externe, liste et table

  1. J'ai repris le compte-rendu HTML créé au TP précédent et j'ai conservé l'en-tête et le dernier exercice, que j'ai renommé Exercice 0. Enfin, j'ai supprimé les informations de style.
  2. J'ai créé une feuille de style externe compte-rendu-tp3.css où j'ai recopié les informations de style. Ensuite, je l'ai utilisé dans le fichier compte-rendu-tp3.html

    <link href="compte-rendu-tp3.css" rel="stylesheet" media="all" type="text/css">

  3. J'ai crée la classe romain qui utilise les chiffres romains en minuscules pour les listes ordonnées.

    ol.romain {
     list-style-type: lower-roman;
    }

  4. J'ai créé une table HTML basique dans mon compte-rendu, en utilisant des balises table, tr, th, td.
  5. J'ai affecté la valeur 100% à la proprièté width dans la classe table afin que la table occupe toute la largeur du compte rendu.
  6. J'ai crée une classe qui s'appelle titre pour la balise tr pour centrer les titres et utiliser des petites majuscules.

    tr.titre {
     font-variant: small-caps;
     text-align: center;
    }

  7. Pour aérer la table avec la propriété padding, j'ai crée une classe commune pour les éléments table, th ,td.

    table, th, td {
     padding: 10px;
    }

  8. Pour alignéer les valeurs du tableau à gauche, sauf celles de la dernière colonne qui doivent être centrées, j'utilise la classe .centered qu'avec la dernière colonne.
  9. Voilà, comment j'ai changé la couleur de fond des lignes paires.

    tr:nth-child(even) {
     background-color: rgb(80%, 80%, 90%);
    }

  10. Codant en hexadécimal, 28 = 1*16+12 = 1C, 120 = 7*16+8 = 78, 12 = 0*16+12 = 0C. D'où j'ai affecté la valeur #1C780C à la propriété background- color dans la pseudo-class tr:hover.

Exercice 2 : La ferme des animaux

  1. J'ai téléchargé le fichier zippé et je l'ai extrait dans le dossier tp3.
  2. J'ai crée une classe img où j'ai redimensionné les images en 180px.
  3. J'ai utilisé l'attribut float: left dans la classe div.
  4. Si on fait varier la taille de la fenêtre du navigateur horizontalement, les images retournent à la ligne à gauche lorsque la fenêtre devient de plus en plus petite.
  5. Oui, je peux aussi mettre toutes les images à droite en changeant juste float: left avec float: right dans la partie CSS du document.
  6. Dans la classe .animal que j'ai créé, j'ai utilisé les attributs width, text-align, color, background, padding-top, padding-bottom en sorte que les titres (animal1, animal2, etc.) soient affichés en blanc sur fond noir et soient centrés horizontalement avec un espacement convenable.
  7. J'ai ajouté la propriété margin-left: 10px; à la classe img afin d'espacer les unes des autres les six boîtes contenant les images.
  8. J'ai ajouté la propriété border: 3px solid #aaaaaa; à la classe img pour ajouter une bordure de trois pixels de largeur de couleur #aaaaaa aux images à l'intérieur des boîtes.
  9. J'ai réduit l'opacité à 50% lorsque l'utilisateur passe sa souris au-dessus d'une image dans une boîte.

    img:hover {
     opacity: 0.5;
    }

  10. Pour faire en sorte que les titres soient accolés par le bas à l'image et mordent légèrement sur l'image, j'ai utilisé la propriété position, top, right, clear.
  11. J'ai mis le texte à mettre après en-dessous des images et de leurs titres avec la classe end.

    .end {
     clear: left;
     text-align: left;
    }

Exercice 3 : Barre de menu

  1. J'ai fixé la position de la liste en haut à gauche de la partie affichage du navigateur avec les propriétés position: fixed; top: 0 ; left: 0;.
  2. J'ai retiré le style par défaut des items de liste avec list-style- type: none;. dans la classe .menu des éléments ul.
  3. J'ai affiché les items en gras et avec une police Arial (ou sinon sans -serif) avec les propriétés font-weight: bold; font-family: "Arial", "Helvetica", "Verdana", sans-serif; dans la classe .barre des éléments li.
  4. Dans le but d'afficher les items horizontalement collés à gauche et non plus verticalement, j'ai ajouté la propriété align-items : left ; dans la classe ul.menu.
  5. J'ai mis width: 100px; padding-right: 10px; de façon que les items aient une largeur de 100 pixels et un espacement d'une dizaine de pixel entre eux.
  6. J'ai introduit la propriété background: purple; de manière de changer le fond du menu pour qu'il soit de couleur violette dans la classe de l'élément ul. Quant à la couleur des items, j'ai intoduit la propriété color: white; dans la classe de l'élément li.
  7. J'ai fixé la hauteur du menu à 50 pixels puis j'ai fait en sorte que le menu occupe toute la largeur en utilisant les propriétés height: 50px; width: 100%; dans la classe ul.menu.
  8. J'ai ajouté un style à l'item About pour l'afficher à droite de la barre de menu.

    <li class="barre" style="float: right;">About</li>

  9. J'ai mis une bordure de 2 pixels seulement en bas (border-bottom: 2px solid black;) pour afficher une ligne de 2 pixels en bas du menu.
